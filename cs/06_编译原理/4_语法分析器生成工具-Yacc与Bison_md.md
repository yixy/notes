# 语法分析器生成工具-Yacc与Bison #

Flex就是由Vern Paxon实现的一个Lex，Bison则是GNU版本的YACC。

## 1 Yacc ##

Yacc会生成一个叫做『语法分析器』的程序。它接收词法分析器生成的token，转换为抽象语法树AST。

```
example.y -->【Yacc编译器】 --> lex.yy.c --> 【C编译器】 --> a.out

词法单元序列  --> 【a.out】(语法分析器) --> 抽象语法树AST
```
