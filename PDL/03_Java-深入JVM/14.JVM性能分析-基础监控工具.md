```
4007 gradle-language-server.jar
4345 GradleServer
4315 sonarlint-ls.jar
4412 BootLanguageServerBootApp
```

jconsole是更为强大的JVM图形化监控工具，支持本地和远端JVM进程监控。

```
jconsole
```
